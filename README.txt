# JW Pressure Washing Website

This is a simple static website ready for GitHub Pages.

## Files
- `index.html`
- `styles.css`
- `logo.png`

## How to publish for free with GitHub Pages

### Option 1: Create a repository named `jw-pressure-washing`
1. Log in to GitHub.
2. Create a new public repository named `jw-pressure-washing`.
3. Upload `index.html`, `styles.css`, and `logo.png`.
4. Go to **Settings** → **Pages**.
5. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main`
   - **Folder:** `/root`
6. Save.
7. After a minute or two, your site should be live.

Your URL will usually look like:
`https://YOUR-GITHUB-USERNAME.github.io/jw-pressure-washing/`

### Option 2: User site
If you want the cleanest URL possible:
1. Create a public repository named `YOUR-GITHUB-USERNAME.github.io`
2. Upload the files there
3. Enable GitHub Pages

Then your site URL becomes:
`https://YOUR-GITHUB-USERNAME.github.io/`
